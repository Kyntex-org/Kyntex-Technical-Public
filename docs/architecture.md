# Architecture

This document provides a high-level overview of the Kyntex project structure.

## Public Layer

- Website: kyntex.org
- Public documentation: Kyntex-Technical-Public
- Public GitHub organization: Kyntex-org

## Private Layer

- Internal development repositories
- Draft technical notes
- Experimental code
- Project planning materials

## Design Goal

The project separates public-facing documentation from internal development work so that public materials remain clean, understandable, and safe to share.
